# Sistem Pelaporan Kerusakan Fasilitas Kampus

Aplikasi berbasis C# WinForms yang digunakan untuk melaporkan kerusakan fasilitas kampus dan membantu admin dalam mengelola status laporan.

## Fitur

- Dashboard informasi laporan
- Input laporan kerusakan
- Upload foto kerusakan
- Login admin
- Kelola status laporan
- Rekap data laporan

## Teknologi

- C# WinForms
- .NET 10
- Penyimpanan data menggunakan file `.txt`

## Akun Admin

- **Username:** admin
- **Password:** 12345

## Cara Menjalankan

1. Buka project menggunakan Visual Studio.
2. Jalankan aplikasi.
3. Pengguna dapat langsung membuat laporan tanpa login.
4. Admin login untuk mengelola status laporan dan melihat rekap.
